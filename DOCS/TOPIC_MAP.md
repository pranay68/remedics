# Topic Map (Pillars + Clusters)

This is the **information architecture** for topical authority: pillars target the big queries; clusters target long‑tail queries and link back to their pillar.

Rules
- Every cluster page links → its pillar.
- Every pillar links → 5–20 clusters.
- Comparison pages always cross‑link to other comparisons and to the relevant pillar.
