# Aternox ReArch Public Site

Public company site for `aternox.site`, centered on ReArch recovery.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Current site shape

- Aternox is the parent company identity.
- ReArch is the public product story.
- Legacy engine, compare, and research routes are redirected to the new information architecture.
